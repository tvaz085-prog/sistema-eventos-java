# Sistema de Eventos em Java

Sistema desenvolvido em Java para cadastro e gerenciamento de eventos.

Funcionalidades:
- Cadastro de usuário
- Cadastro de eventos
- Participação em eventos
- Cancelamento de participação
- Filtro por cidade
- Ordenação por data
- Identificação de status (passado, atual, futuro)
- Persistência em arquivo

Projeto desenvolvido para atividade acadêmica.
